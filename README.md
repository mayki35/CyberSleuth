### Instalation on Linux
```
git clone https://github.com/mayki35/CyberSleuth.git
```
```
cd GhostTrack
```
```
sudo su
```
```
python -m venv venv
```
```
sourse .\venv\bin\activate
```
```
pip install -r requirements.txt
```
```
python CyberSleuth.py
```
### Instalation en Windows
```
cd GhostTrack
```
```
python -m venv venv
```
```
.\venv\Scripts\activate
```
```
pip install -r requirements.txt
```
```
python CyberSleuth.py
```
